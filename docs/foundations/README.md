## Rule

Each foundation lives as a single Markdown file at `docs/foundations/{surface}/{slug}.md`. Complete foundations keep exact proof in `docs/foundations/{surface}/{slug}.evidence.md`. Surface folders contain their `README.md` hub, foundation files, and matching evidence sidecars only.

Do not add placeholder foundation files.
