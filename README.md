## Local dev

```bash
python scripts/axis.py local-dev up
```

Open the web app at <https://localhost:3000>. For ports, HTTPS setup, troubleshooting, and observability, see [docs/playbooks/local-dev.md](./docs/playbooks/local-dev.md).
